# Admin Guide: Managing Background Media

## Quick Start

### Step 1: Access Admin Panel
1. Login with admin credentials: `admin@properestate.com`
2. Click on your profile icon
3. You'll be redirected to the Admin Dashboard

### Step 2: Navigate to Backgrounds
1. In the Admin Panel sidebar, click **🎬 Backgrounds**
2. You'll see the Background Media Management page

### Step 3: Upload New Background

#### What You Can Upload:
- **Videos**: MP4, WebM, MOV files (recommended for cinematic effect)
- **GIFs**: Animated GIF files (great for subtle animations)
- **Images**: PNG, JPEG files (static backgrounds)

#### How to Upload:
1. Click the file input under "Upload New Background"
2. Select your video/GIF/image file
3. Wait for "Uploading..." message to complete
4. Your media will appear in the grid below

#### Best Practices:
- **Video Resolution**: 1920x1080 (Full HD) or higher
- **File Size**: Keep under 50MB for faster loading
- **Duration**: 10-30 seconds for videos (they loop automatically)
- **Content**: Choose dark, subtle backgrounds that don't distract from text
- **Aesthetic**: Match the cinematic, premium feel of the site

### Step 4: Manage Backgrounds

#### Activate/Deactivate
- **Green "ACTIVE" badge**: Background is currently showing on home page
- **Red "INACTIVE" badge**: Background is hidden
- Click **Activate** to show a background
- Click **Deactivate** to hide a background
- Multiple backgrounds can be active at once (they rotate every 10 seconds)

#### Delete Background
- Click the **Delete** button to permanently remove a background
- Confirm the deletion in the popup
- The file will be deleted from the server

### Step 5: Preview on Home Page
1. Open the home page in a new tab
2. Watch the backgrounds rotate every 10 seconds
3. Smooth fade transitions between backgrounds
4. Backgrounds enhance the hero section

## Tips for Best Results

### Cinematic Aesthetic
- Use slow-moving aerial shots
- Dark, moody lighting
- Architectural or landscape footage
- Luxury property videos
- Abstract motion graphics

### Performance
- Compress videos before uploading (use HandBrake or similar)
- Test on different devices
- Keep total active backgrounds under 5 for best performance
- Use WebM format for better compression

### User Experience
- Don't use distracting or fast-moving content
- Ensure text remains readable over backgrounds
- Test with different screen sizes
- Consider seasonal themes (monsoon, festivals, etc.)

## Troubleshooting

### Upload Not Working
- Check file size (must be under 100MB)
- Verify file format (MP4, WebM, GIF, PNG, JPEG)
- Ensure you're logged in as admin
- Check browser console for errors

### Background Not Showing
- Make sure background is marked as **ACTIVE**
- Refresh the home page
- Clear browser cache
- Check if file uploaded successfully

### Slow Loading
- Compress video files before uploading
- Reduce video resolution to 1080p
- Use WebM format instead of MP4
- Limit active backgrounds to 3-4
