## 🐛 Common Issues & Quick Fixes

### Issue: Styles not loading
**Fix:** Hard refresh (Ctrl+Shift+R or Cmd+Shift+R)

### Issue: Animations not working
**Fix:** Check browser console (F12) for errors

### Issue: Fonts look wrong
**Fix:** Wait a few seconds for Google Fonts to load

### Issue: Video background not showing
**Fix:** This is expected — gradient fallback is shown by default

### Issue: Page looks broken
**Fix:** Make sure you're in the `frontend` folder when running `npm start`
